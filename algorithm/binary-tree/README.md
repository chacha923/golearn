## 递归
> 大部分二叉树问题都用递归思维


### 递归序
深度遍历，每个节点要被访问 3 次


### DFS
尽可能搜到底，用栈，递归


### BFS
尽可能横向搜，用队列




## 非递归

### 不能只会递归啊！！

任何递归都可以改成非递归实现
> 递归不是玄学，只是系统帮你压栈  -- 左神

所以说，非递归就是考察手写栈，也可以看做最基本的回溯思想
比较少见的，有时也考察滑动窗口什么的技巧
